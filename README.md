# einfra
This is monorepo project for managing infrastructure application. The project is divided into multiple packages, each responsible for a specific aspect of infrastructure management. The packages are:

- `security_app`: This package contains the application for managing security infrastructure.
- `network_app`: This package contains the application for managing network infrastructure.
- `storage_app`: This package contains the application for managing storage infrastructure.
- `compute_app`: This package contains the application for managing compute infrastructure.
- `monitoring_app`: This package contains the application for managing monitoring infrastructure.
- `logging_app`: This package contains the application for managing logging infrastructure.
- `database_app`: This package contains the application for managing database infrastructure.
- `infrastructure_app`: This package contains the application for managing infrastructure infrastructure.
